C Language

"C" is a general purpose high-level programming language.


It is a procedural oriented programming language.


It is a machine independent language[not platform independent].


It is the mother language of modern programming languages.


It is used to develop system softwares,application softwares, and compilers etc.


It was developed by "Dennis Ritchie" in 1972 at T-bell laboratory.


It was develop to make an operating system called "Unix".


It is a case-sensetive programming language, that supports lower case[use small letters during coding].


Note: -1


Machine independent means it allows developers to develop programs without worrying about the hardware platforms where they will be implemented.



Note: -2



Unix is one of the most popular network operating systems in use today and the heart of the Internet data superway.
